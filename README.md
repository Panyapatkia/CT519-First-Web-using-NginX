รายละเอียดไฟล์

index.html : หน้าเว็บหลัก

README.md : เอกสารอธิบาย Project

โครงงานนี้เป็นการศึกษาการสร้าง Web Server อย่างง่ายด้วย Docker และ Nginx โดยพัฒนาเว็บเพจแบบ Static HTML พร้อมสร้าง Docker Image และเผยแพร่ Image ไปยัง Docker Hub รวมถึงจัดเก็บ Source Code บน GitHub
วัตถุประสงค์ของโครงงาน คือ
- ศึกษาการใช้งาน Docker Container
- ศึกษาการทำงานของ Nginx Web Server
- สร้าง Docker Image จาก Dockerfile
- Deploy เว็บไซต์ด้วย Docker Container
- Push Docker Image ไปยัง Docker Hub
- จัดเก็บ Source Code บน GitHub Repository
